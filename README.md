# Persona Persist / Prestige Concierge

Persona Persist is a luxury concierge web application built with React, TypeScript, Vite, Tailwind CSS, and Lucide React icons. The app provides a premium digital experience for private members to access concierge services such as flight booking, dining reservations, hotel booking, chauffeur services, shopping, security, VIP events, and Ultra-tier premium privileges.

## Live Demo

https://ikramkofficial.github.io/persona-assist/

## Project Overview

This project is designed as a premium concierge dashboard with a modern iOS-style glassmorphism interface. Users can create a private profile, sign in using a secure PIN, link a payment card, manage wallet balance, book luxury services, chat with Persona Assist, track activity, receive dynamic notifications, and unlock Persona Ultra services using access keys.
